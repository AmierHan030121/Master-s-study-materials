实验一：中文文本分析与可视化——以“今日头条”汽车新闻为例

一、 实验目的

1. 掌握基础知识：熟悉自然语言处理（NLP）的基本概念和标准流程，包括文本获取、预处理、分词、词频统计和结果可视化。

2. 实践工具应用：熟练使用 pandas 进行数据读取与处理，掌握 jieba 库进行中文分词，并利用 matplotlib 和 wordcloud 库实现数据可视化。

3. 培养分析能力：通过对特定领域（汽车新闻）的文本数据进行分析，学习如何从非结构化文本中提取有价值的信息和洞察，例如识别热门话题、关注焦点等。

4. 理解停用词作用：通过对比去除停用词前后的分析结果，深刻理解文本预处理在NLP任务中的重要性。

    二、 实验环境

    • 编程语言：Python 3.x
    • 开发环境：Jupyter Notebook 或 VS Code (with Python extension)
    • 必需库：
      • pandas: 用于读取和处理Excel数据。
      • jieba: 用于高效的中文分词。
      • matplotlib: 用于绘制词频统计图表。
      • wordcloud: 用于生成词云图。
      • numpy & Pillow(PIL): 用于处理词云图的背景蒙版。

    实验开始前，请确保已安装所有必需的库。 你可以在Jupyter Notebook中运行以下命令来安装：

    !pip install pandas jieba matplotlib wordcloud numpy Pillow

    三、 实验数据与资源

    1. 数据集：今日头条-汽车.xlsx - 包含今日头条平台上关于汽车新闻的标题和内容。

    2. 停用词表：stopwords.txt - 一个包含中文常用停用词（如“的”、“是”、“在”等）的文本文件。你可以使用你提供的文件，或者从网上下载一份（例如哈工大停用词表）。

    3. 背景蒙版：car.png - 一张汽车形状的白色背景、黑色图案的图片，用于生成特定形状的词云图。

      请将以上文件与你的Jupyter Notebook文件放在同一目录下或指定正确的路径。

              def generate_wordcloud(word_counts_dict, background_shape_path=None):
    """
    根据词频字典生成词云图。
    
    参数:
    word_counts_dict (dict): 词语为键，频率为值的字典。
    background_shape_path (str, optional): 蒙版图片的路径. 默认为None.
    """
    mask = None
    if background_shape_path:
        mask = np.array(Image.open(background_shape_path))

    wordcloud = WordCloud(
        font_path='C:/Windows/Fonts/msyh.ttc', # 请确保字体路径正确
        background_color="white",
        width=1000,
        height=600,
        mask=mask,
        contour_width=3, 
        contour_color='steelblue',
        collocations=False # 避免词语重复显示
    )
    
    wordcloud.generate_from_frequencies(word_counts_dict)
    
    plt.figure(figsize=(12, 8))
    plt.imshow(wordcloud, interpolation='bilinear')
    plt.axis("off")
    plt.show()

              注意：font_path 需要设置为你计算机上存在的中文字体文件路径。Windows系统通常在 C:/Windows/Fonts/ 目录下，msyh.ttc（微软雅黑）是一个不错的选择。
